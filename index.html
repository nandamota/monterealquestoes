🔥 Configurar Firebase
1Acesse console.firebase.google.com e crie um projeto.
2No projeto, vá em Firestore Database → criar banco → modo produção.
3Vá em Configurações do projeto → Seus apps → Adicionar app Web → copie o objeto firebaseConfig.
4Em Firestore → Regras, cole: allow read, write: if true; (para uso pessoal).
4bPara a Biblioteca: vá em Storage → Começar → modo produção. Em Storage → Regras, cole: allow read, write: if true;
5Cole o JSON de configuração abaixo e salve.

🧠 Como o sistema aprende

Toda questão marcada como "no banco" (estrela dourada) vira um exemplo real enviado à IA antes de cada geração. O sistema busca automaticamente até 3 questões do mesmo tipo e série e as usa como modelo de estilo, vocabulário e formato.

Quanto mais questões boas o professor salvar no banco, mais preciso e personalizado o sistema fica ao longo do tempo.
